# Command Reference

## `aw --help`

Shows the Artworks command surface.

## `aw version`

Prints the current CLI version.

## `aw doctor`

Runs a lightweight environment check.

## `aw studio init`

Initializes a studio workspace placeholder.

## `aw project new [NAME]`

Creates a production placeholder. If no name is provided, the command uses `UNTITLED`.

## `aw project open [NAME]`

Opens an existing production placeholder. If no name is provided, the active production is implied.

## `aw docs bootstrap`

Bootstraps production documentation placeholders.

## `aw docs validate`

Validates production documentation placeholders.

## `aw git sync`

Synchronizes production history placeholders.
